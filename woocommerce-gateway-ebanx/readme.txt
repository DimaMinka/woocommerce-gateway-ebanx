== Description ==

The WooCommerce EBANX Payment Gateway plugin allows you to accept local & international payment methods directly on your ecommerce store. Using the EBANX plugin you can process relevant cash, online debit, credit card, and EBANX Wallet payments in Brazil, Mexico, Colombia, Chile & Peru. **No technical knowledge is needed for installation.** Installation is the simple, the way it should be.

**Exclusive Advantages**

* Security is already taken care of, the customer’s sensitive data does not goes to your servers and is saved in EBANX environment using PCI standards
* One-click purchases which allow your client to skip the checkout process
* Checkout payment form is responsive and adapts nicely to all mobile screen sizes and to every theme
* Everything you need in one plugin, you don’t have to install any external plugins or extensions

**Customize and Manage Your Payments**

With the EBANX plugin, you can:

* Choose which payment methods are displayed at checkout 
* Set a maximum number of installments 
* Select an expiration date for cash payments
* Allow customers to save their credit card information
* Create orders directly in WooCommerce 
* Request refunds directly in WooCommerce

The plugin also includes:

* Sandbox mode for testing 
* Capture mode that when activated allows you to collect payments after a manual review 
* Extra fields that are added automatically for payments made in Brazil or Chile where customers must provide more information to local regulatory authorities

Looking for more detailed information? Visit our [Developer’s Academy](https://www.ebanx.com/business/en/developers/integrations/extensions-and-plugins/woocommerce-plugin "EBANX Developer's Academy") for step-by-step guides, API references, integration options, and more. 

**Requirements**

All pages that incorporate the EBANX plugin must be served over HTTPS.

== Installation ==

**Automatic**

Automatic installation is the easiest option and can be done without leaving your web browser. To do an automatic install of the EBANX plugin, login to the WordPress Dashboard, go to the Plugins menu, and select “Add New.” Then, search for the “EBANX Gateway Plugin” and click “Install Now.”

**Manual**

To install the plugin manually, download our plugin and upload it to your web server via an FTP application. Visit the [WordPress codex](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation "WordPress codex") for further instructions. 

**Via Github**

The EBANX Gateway Plugin can also be installed using GitHub. You can find our repository [here](https://github.com/ebanx/checkout-woocommerce/ "EBANX GitHub repository"). To view step-by-step installation via GitHub instructions click [here](https://help.github.com/articles/cloning-a-repository/ "GitHub"). 

Still need your API Keys? You can pick those up [here](https://www.ebanx.com/business/en/dashboard "EBANX API keys").

= Which payment types does EBANX process? =

* Visa, Mastercard, American Express, Diner’s Club, Discover - all countries
* EBANX Boleto, Cash Payment - Brazil
* Hipercard, Elo, and Aura Domestic Credit Cards - Brazil
* Online Debit Transfer - Brazil 
* EBANX Wallet, Prepaid Card / Debit Transfer - Brazil
* OXXO, Cash Payment - Mexico
* Debit & Credit Cards - Mexico
* EBANX Wallet, Debit Transfer - Mexico
* Pagos Seguros en Línea (PSE), Online Debit Transfer - Colombia
* Sencillito, Cash Payment - Chile
* Servipag, Online Debit Transfer - Chile
* PagoEfectivo, Cash Payment / Debit Transfer - Peru
* SafetyPay, Cash Payment / Debit Transfer - Peru
